# CUP_TO_LX7000

This is an application to convert data from the .cup file format and use them in LXe and import them as user airports.
This application is built using Qt 6.10

# HOW TO USE

1. Get the Qt Installer from: https://www.qt.io/development/download-qt-installer-oss
2. Install Qt
3. Download source code for CUP_TO_LX7000
4. Open CMakeList.txt to open the project in Qt Creator
5. Build

## LXe

1. Open any database in LXe
2. Import the exported file from CUP_TO_LX7000
3. ONLY select USER under States
4. Write to device
